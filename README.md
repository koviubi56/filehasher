# Filehasher

Filehasher can hash a file, and compare the user's hash and the file's hash.

This is useful, if you don't want to get hacked.

Hash algorithms currently supported:

* MD5
* SHA1
* SHA256
* SHA512
* SHA3 512
* Blake2s
* SHA384
* SHA3 256
* SHA3 224
* Blake2b
* SHA3 384
* SHA244
